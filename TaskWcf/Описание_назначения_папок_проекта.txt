Application/BL

Классы для реализации бизнес логики

Common

Вспомогательные классы для бизнес-логики. Хелперы, конвертеры и т.д.

Extentions

Вспомогательные классы для реализации бизнес-логики

Infrastructure

Содержит классы для взаимодействия с базой данных. В частности IRepository

Model

Содержит POCO- классы. Они же сущности предметной области. В частности это могут быть сущности code-first. Может содержать модель edmx для подключения к БД
